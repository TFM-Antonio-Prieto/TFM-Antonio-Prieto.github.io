1. Ejemplo de control de errores

La letra del NIF

Los NIF tienen 8 cifras y una letra.

¡Esa letra es un código de control de errores!

Se divide el Nº DNI entre 23 y se haya el resto

La cadena de control es: «TRWAGMYFPDXBNJZSQVHLCKE»

Se haya la letra de la cadena de control que está en la posición correspondiente al resto (empezando a contar por cero)

Chuletario

  • Un servicio de la capa de enlace de datos es el control de errores.
  • Se encarga de detectar (y a veces corregir) los errores que se han producido en la transmisión de una trama.
  • ¿Cómo funciona?
    • El emisor hace una operación matemática sobre la trama a transmitir y manda el resultado junto con la trama.
    • El receptor hace la misma operación y comprueba que el resultado es el mismo que el que el emisor ha enviado.
  • La mayoría de los código pueden detectar un error por trama.
  • Cuanto menos segura es la transmisión más complejo es el sistema de control de errores.

Bit de paridad

Lea y complete

Un sistema control de errores que añade un bit al final de cada byte se llama Bit de paridad.

Si el número final de JXUwMDJkJXUwMDFiJXUwMDAxJXUwMDFj es par se llama paridad par.
Si el número final de unos es impar se llama JXUwMDI4JXUwMDExJXUwMDEzJXUwMDFiJXUwMDBkJXUwMDA1JXUwMDA1 impar.

Habilitar JavaScript

El bit de paridad es uno de los sistemas más simples que existe.
Si se emplea la paridad bidimensional entonces puede corregir un error por cada bloque.
